This application was designed using a worksheet from Dialectical Behavioral Therapy (DBT) which suggests a set of skills for an individual to use based on their distress level. DBT is a therapeutic approach which teaches healthy coping behaviors and living in the moment.

Character Stat Generator

This fun application uses multiple data types and ES6 classes to randomly generate stats for RPG characters. The function which determines the stat numbers simulates a common method for determining stats in which a player rolls 4 d6 dice, drops lowest value, and then adds the remaining values
