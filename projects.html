Character Sheet Generator

Randomly generate character stats for RPG characters. Select World of Warcraft or Dungeons and Dragons and get playable Race/Class combinations for each. The stats are generated using a function which simulates rolling 4 d6, dropping lowest number, and adding the remaining numbers (a common method used for generating stats). The "characters" are generated using javascript classes.
